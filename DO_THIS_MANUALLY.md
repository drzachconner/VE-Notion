### 📊 **Step 1: Add "My Tasks" Database View**

1. Open the dashboard
2. Scroll to **"✅ My Tasks"** section
3. You'll see a gray callout that says "Manual Step Required"
4. **Delete** that callout block
5. On a new line, type: `/linked`
6. Select **"Create linked database"**
7. Search for and select your **"Tasks"** database
8. Click inside the new database view
9. Click **"Filter"** at the top
10. Add filter: **"Assigned To"** → **"Contains"** → **[Team Member Name]**
11. Done! Tasks now show only for this person

## 🆘 Need Help?

If you get stuck on any step:

1. **For linked database views:**
   - Make sure the database exists and is shared with you
   - Type `/linked` not just `/database`
   - Filters are case-sensitive

2. **For creating new databases:**
   - Type `/database` → "Database - Full page"
   - Add properties by clicking "+" button
   - Share with team members who need access

3. **Can't find a database?**
   - Check your "Databases Backend" page
   - Search in Notion sidebar
   - Make sure it's shared with you
