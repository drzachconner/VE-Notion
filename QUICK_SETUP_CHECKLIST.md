## Quick Reference

**How to add navigation links:**
1. Type: `All Tasks | All Projects | All Events`
2. Highlight "All Tasks"
3. Press `Cmd+K` (Mac) or `Ctrl+K` (Windows)
4. Paste URL
5. Repeat for other two

**How to add linked database:**
1. Type `/linked database`
2. Select the database (Tasks, Projects, or Events)
3. Set view type (Table or List)
4. Add filter for the team member
5. Configure columns to show

**How to make bottom links clickable:**
1. Highlight the text (e.g., "View All Tasks")
2. Press `Cmd+K` (Mac) or `Ctrl+K` (Windows)
3. Paste the URL to the team page
